# Project 1: Layoffs Prediction (The Analyst)

## Goal
Move beyond descriptive analytics to predictive modeling. This project aims to predict layoff characteristics based on company attributes.

## Dataset
*   **Source:** Tech Layoffs dataset.
*   **Target:** Predict the magnitude of layoffs or classification (High/Low impact).

## Tech Stack
*   **Language:** R
*   **Core Libraries:** `tidyverse` (Data Cleaning), `tidymodels` (Machine Learning).

## Key Skills
*   Data Cleaning & Preprocessing (recipes).
*   Exploratory Data Analysis (EDA).
*   Model Building (Logistic/Linear Regression).
*   Model Evaluation (Accuracy, RMSE).

# Project 4: MyThemes Package (The Architect)

## Goal
Refactor existing custom visualization scripts into a portable, installable R package.

## Dataset
*   **Source:** Code from `custom_visualization/`.

## Tech Stack
*   **Language:** R
*   **Core Libraries:** `devtools`, `usethis`, `roxygen2`, `ggplot2`.

## Key Skills
*   **Package Development:** Understanding `DESCRIPTION`, `NAMESPACE`.
*   **Documentation:** Writing `roxygen2` comments for help pages.
*   **Software Engineering:** Writing clean, exported functions.

## Deliverable
*   An installable package `mythemes` that can be loaded via `library(mythemes)`.

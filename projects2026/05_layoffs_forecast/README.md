# Project 5: Layoffs Forecast (The Forecaster)

## Goal
Analyze temporal trends in layoffs and forecast future numbers.

## Dataset
*   **Source:** Tech Layoffs dataset (Aggregated by Date).

## Tech Stack
*   **Language:** R
*   **Core Libraries:** `fable` (Forecasting), `tsibble` (Time Series Data), `feasts` (Feature Extraction), `lubridate`.

## Key Skills
*   **Time Series Analysis:** Decomposing trends, seasonality, and noise.
*   **Forecasting:** ARIMA, ETS, or Prophet models.
*   **Visualizing Uncertainty:** Plotting confidence intervals.

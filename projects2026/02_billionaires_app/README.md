# Project 2: Billionaires Dashboard (The Developer)

## Goal
Build a standalone, interactive web application to explore wealth distribution globally.

## Dataset
*   **Source:** Forbes Billionaires dataset.

## Tech Stack
*   **Language:** R
*   **Core Libraries:** `shiny`, `bslib` (Modern UI), `plotly` (Interactive Charts), `dplyr`.

## Key Skills
*   **Reactive Programming:** Understanding `input`, `output`, and `reactive()` expressions.
*   **UI/UX Design:** Creating a clean, modern layout using `bslib`.
*   **Interactive Visualization:** Integrating `plotly` with Shiny.

# Project Context: Billionaires Dashboard

## Objectives
*   Create a production-ready Shiny application.
*   Focus on **User Experience (UX)**: The app should be intuitive and visually appealing.
*   Use `bslib` for theming (do not rely on default Shiny styling).

## Instructions
1.  Setup a basic Shiny app structure (ui.R, server.R or app.R).
2.  Implement filters for Industry, Country, and Wealth range.
3.  Create reactive data subsets based on user input.
4.  Render interactive maps and charts using `plotly`.
5.  Deployable structure (ensure all dependencies are clear).
